# MyPortfolio

MyPortfolio is a responsive personal portfolio website built using pure **HTML** and **CSS**. It showcases my profile, skills, education, and projects with a clean and modern UI.

## 🚀 Features

* Fully responsive design
* Sidebar profile with contact details
* Sections: About, Skills, Education, Portfolio
* Built using CSS Grid & Flexbox
* Smooth hover effects and clean layout

## 🛠️ Tech Stack

* HTML5
* CSS3
